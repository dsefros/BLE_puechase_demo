# iOS BLE Payment Integration Kit

This folder contains integration guidance and reference code for integrating `BlePaymentKit` with iOS CoreBluetooth scanning.

## Included content

- `docs/` — practical integration documentation.
- `reference/` — lightweight CoreBluetooth scanner and advertisement mapper reference source.
- `examples/` — integration snippets using the current `BlePaymentKit` API.
- `test-vectors/` — copied at artifact build time from `kit/docs/test-vectors` (shared source of truth).

In the generated archive, the Swift Package source is shipped separately at `BlePaymentKit/`.

Start with:
- `docs/quick-start.md`
- `docs/scan-result-to-sdk-input.md`
